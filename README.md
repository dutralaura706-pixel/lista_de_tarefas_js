#  Lista de Tarefas JS

Olá, sou Laura Dutra 👋

Este é um projeto de uma lista de tarefas desenvolvido para praticar conceitos de desenvolvimento web, principalmente JavaScript e manipulação do DOM.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Organizar atividades do dia a dia

## Como executar

1. Clone este repositório.
2. Abra o arquivo index.html no navegador.
3. Utilize a lista de tarefas.

## Objetivo do projeto

Projeto criado para desenvolver prática em lógica de programação, eventos em JavaScript e criação de interfaces simples.
